summer-game
моя первая сюжетная игра в терминале на python

Как запустить:
1. Скачай файл v1.0 code.py
2. Открой терминал в папке с файлом
3. Напиши: python "v1.0 code.py"

Сюжет:
Ты просыпаешься, идёшь в магазин, выбираешь еду.
В конце понимаешь, что лето прошло незаметно.

- v2.0 — в планах
